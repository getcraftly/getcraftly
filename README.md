## 🧰 What we build

Production-ready templates with:

- **Next.js 16.2** — App Router, Turbopack default, async params
- **Tailwind CSS v4.1** — `@theme inline`, OKLCH colors, config-less
- **TypeScript strict** — zero `any`, fully typed
- **Framer Motion** — polished animations out of the box
- **Dark mode** — proper CSS variables, not hacks
- **Accessible** — focus rings, ARIA, reduced-motion support
